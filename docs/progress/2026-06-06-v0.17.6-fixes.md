# 2026-06-06 — v0.17.6-fixes

> 진행 단계: **배포 — 첫 공개 릴리스 후속 핫픽스**
> 상태: 진행 중

첫 공개 릴리스(v0.17.5)가 Apple Silicon 에서 "손상되어 열 수 없음" 으로 실행 불가 + 인앱 뷰어 아이콘 누락. v0.17.6 으로 재릴리스.

## 완료
- macOS ad-hoc 코드 서명: `electron-builder.yml` 에서 `identity:null` 제거(서명을 건너뛰어 번들이 무효가 되던 원인), afterPack 훅(`scripts/adhoc-sign.cjs`)에서 `codesign --force --deep --sign -` 재서명 → `codesign --verify` 통과. Developer ID 없이 실행 가능 (단 quarantine 은 남아 first-run 우클릭>열기 또는 `xattr -cr` 필요)
- 앱 표시명 `cairn` → `Cairn` (productName)
- 인앱 Notion 뷰어: callout 의 external/file 아이콘 렌더 (기존엔 emoji 만 추출 → Claude 아이콘이 Notion 엔 뜨지만 뷰어엔 누락). `SimpleBlock.iconUrl` 추가
- Liquid Glass: packaged compositor 에서 `backdrop-filter` 가 실제 렌더 안 돼도 패널이 깨져 보이지 않게 inner edge + 그림자로 입체감 보장. 52% 유지, GPU 강제 안 함
- 릴리스 노트 @멘션 escape 를 `&#64;` → ZWSP(U+200B) (GitHub 이 `&#64;` 를 디코드 후 재링크화해 bogus Contributors 가 생기던 문제)

## 시행착오 / 결정
- ad-hoc 서명은 "실행 가능" 의 최소선일 뿐 Gatekeeper quarantine 은 못 없앰 → 완전한 더블클릭 실행은 Developer ID + notarization($99/yr) 필요. 0.x 단계에선 ad-hoc + 우클릭>열기 안내로 감
- Liquid Glass blur 미렌더 원인은 GPU 단정 불가(stacking/compositing 조건 민감, Reddit r/css 참고). GPU 강제는 크래시 리스크라 과한 대응 → blur 를 보너스로 취급하고 CSS 로 기본 면·대비 보장하는 방향
- 첫 릴리스가 깨진 채 배포돼 v0.17.5 는 폐기, v0.17.6 으로 재발행
