## 완료
- **포커스 링 통일** — 전역 `:focus-visible` 없어 Chromium 기본 링이 테마별로 다른 색(주황/파랑)으로 뜨던 것 → accent 링으로 통일.
- **라이트 팔레트 보정** — 캔버스 순백(#fff)→부드러운 그레이-화이트(눈부심↓), 카드(surface-1) 더 밝게 띄워 계층 또렷, hairline 진하게(멍함↓), ink 더 진하게(대비↑).
- **라이트 글래스 투명도 복원** — #156 에서 너무 불투명(74~84%)하게 만든 것 → 흰색 베이스 58~70% 로 낮춰 뒤가 비치는 유리감.
- **테스트 알림 인앱 피드백** — macOS 는 앱이 앞이면 배너 안 띄워 무반응처럼 보이던 것 → "보냈어요" + 안내 문구(설정 확인)·미지원 처리.
- **date-picker 위치** — 모달 안에서 팝오버가 버튼을 가리던 것 → body portal + 고정 위치(아래 공간 없으면 위로 flip).
- **드로어 본문 대비** — 콜아웃은 밝고 본문은 흐릿하던 것 → 본문 text-ink 로 또렷하게.
