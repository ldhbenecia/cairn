## v0.16 — 디자인 본격 + Preferences (사용자 명시)

### 16.1 — UI 본격 polish (sidebar / RunPanel 등)

사용자 지적: "UI 가 좀 별로같은데" / "공간 활용 별로". 14.6 / v0.15.2 에서 시작했지만 본격은 아직.

- Sidebar nav hover / active 시각 정교화 (transition / 좌측 indicator bar 등)
- RunPanel = action area + status + result + (mini list daily / placeholder weekly·monthly) 의 spacing / typography 정돈
- 결과 카드 = Linear 의 detail-pane 톤 (헤더 / meta / link / action)
- 트레이 brand 아이콘 PNG (`io.cairn.desktop`) — 현재 텍스트 'cairn'

### 16.2 — Preferences panel + 다크/라이트 모드

- Settings → Preferences (또는 sidebar 의 새 항목)
- 테마 토글 (system / dark / light)
- styles.css 에 `:root` (다크) + `@media (prefers-color-scheme: light)` 또는 명시 `[data-theme="light"]` 토큰 fork
- light mode 토큰 = storm slate 의 어두운 변종 (canvas #f7f8f8, accent #2a3a4c 등)
- 별도 ADR (`0015-light-mode-tokens` 후보)

### 16.3 — config.json 셋업 튜토리얼

- 사용자 본인이 cairn 처음 셋업 시 노션 페이지 등록 단계 단계별 안내
- 형식: 인앱 step indicator (welcome → notion integration token → 부모 page picker → github token → repo picker → 검증)
- v0.2 셋업 마법사 영역과 합쳐서 같이 (v0.16 의 16.3 = v0.2 셋업 마법사의 시작?)

## v0.17 — 사용자 prompt 커스텀

사용자 명시: "지금 작업일지 프롬포트 내가 정의해놓은거 쓰는데 이것도 사용자 커스텀 할 수 있게 할거고".

- cairn engine 의 hard-coded prompt (현재 `DailySummarizerService` / `RollupSummarizerService` 안에 있는 system prompt) → 사용자 정의 가능
- Preferences 안 "Summarizer prompt" 섹션:
  - 모드별 (daily / weekly / monthly) prompt template 편집기 (multi-line text)
  - 변수 토큰 ($date, $activities, $sourceCounts 등) 정의
  - cairn engine 이 worklog.config.json 또는 별도 `~/.cairn/prompts/` 에서 사용자 prompt 우선 load + fallback 기본
  - 토큰 사용량 추정 미리보기 (선택)
- 별도 ADR (`0016-user-defined-prompts` 후보)
- engine 변경 분량 큼 (PR 시리즈 17.1 ~ 17.x)
